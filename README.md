# vazo-stream

Next.js app-router starter for Vazo Stream (Ghost multi-embed view-multiplication).

## Setup
1. Copy `.env.example` to `.env.local` and fill values.
2. Install dependencies: `npm install`
3. Run in dev: `npm run dev`
4. Build: `npm run build`

## Deploy
- Deploy to Vercel and set environment variables in the project settings.
- After deploy, optionally run the GitHub Action `deploy-warm` to warm fragment caches.
